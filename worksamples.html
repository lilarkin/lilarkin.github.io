Civis

A web and Native application allowing users access to legislators including House Representative by congressional district using anaddress form or geolocation. User can access individual legislator’s contact info, bio, and recent news articles. Some of my specific roles include:
  • Write API Users controller spec testing.
  • Bring in Wikipedia API gem to extract legislators’ biographies. Create custom rake task that ensures bio belongs to legislator to avoid duplicate names and handle edge cases.

Technologies Used: Ruby, Ruby on Rails, ReactJS, Redux, Rspec, Devise, Materialize. Incorporate 5 APIs: Propublica, New York Times, Wikipedia, Google (maps and civic info), Twitter
